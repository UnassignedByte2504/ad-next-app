> **Version:** 1.0.0

## Brand Philosophy

### Concepto: "Magia Profesional"

Ayla Designs crea un puente entre lo místico y lo profesional. Diseños que capturan la esencia bohemia - elementos celestiales, tonos terrosos, formas orgánicas - sin sacrificar la elegancia corporativa que las marcas modernas necesitan.

**Valores de marca:**
- **Magia** - Diseños que inspiran y evocan emociones
- **Profesionalidad** - Estética refinada y versátil
- **Autenticidad** - Cada plantilla tiene alma propia
- **Accesibilidad** - Fácil de usar, fácil de personalizar

### Personalidad

| Atributo    | Expresión                               |
| ----------- | --------------------------------------- |
| Mística     | Elementos celestiales, lunas, estrellas |
| Cálida      | Paleta amber/dorada, tonos tierra       |
| Elegante    | Tipografía serif, espaciado generoso    |
| Bohemia     | Formas orgánicas, texturas naturales    |
| Profesional | Layouts limpios, jerarquía clara        |

### Tono de Voz

- **Inspirador** pero no cursi
- **Cercano** pero profesional
- **Femenino** sin ser exclusivo
- **Místico** sin ser esotérico

Ejemplos:
- ✅ "Diseños que dan vida a tu visión"
- ✅ "Plantillas con alma bohemia"
- ❌ "Los mejores diseños del universo"
- ❌ "Magia mística para tu negocio espiritual"

- **60%** - Stone neutrals (backgrounds, superficies)
- **30%** - Amber/Gold (elementos principales)
- **10%** - Purple/Lavender (acentos, highlights)

## Typography

### Font Stack

**Primary Font: Cormorant Garamond**
- Uso: Headings, títulos de productos, elementos destacados
- Características: Elegante, clásica con toque moderno
- Weights: 400 (Regular), 500 (Medium), 600 (SemiBold), 700 (Bold)
- Incluye: Italic para énfasis elegante

**Secondary Font: Nunito Sans**
- Uso: Body text, párrafos, formularios, UI
- Características: Redondeada, amigable, alta legibilidad
- Weights: 300 (Light), 400 (Regular), 500 (Medium), 600 (SemiBold)

**Monospace: JetBrains Mono**
- Uso: Precios, códigos de descuento, datos técnicos
- Weight: 400

### Type Scale

```css
/* Headings - Cormorant Garamond */
--font-h1: 700 4rem/1.1 'Cormorant Garamond';      /* 64px - Hero */
--font-h2: 600 3rem/1.2 'Cormorant Garamond';      /* 48px - Section titles */
--font-h3: 600 2.25rem/1.3 'Cormorant Garamond';   /* 36px - Subsections */
--font-h4: 500 1.5rem/1.4 'Cormorant Garamond';    /* 24px - Card titles */
--font-h5: 500 1.25rem/1.4 'Cormorant Garamond';   /* 20px - Small titles */
--font-h6: 500 1rem/1.5 'Cormorant Garamond';      /* 16px - Captions */

/* Body - Nunito Sans */
--font-body-lg: 400 1.125rem/1.7 'Nunito Sans';    /* 18px */
--font-body: 400 1rem/1.6 'Nunito Sans';           /* 16px */
--font-body-sm: 400 0.875rem/1.5 'Nunito Sans';    /* 14px */
--font-caption: 300 0.75rem/1.4 'Nunito Sans';     /* 12px */

/* UI Elements */
--font-button: 600 0.875rem/1 'Nunito Sans';
--font-label: 500 0.75rem/1 'Nunito Sans';
--font-price: 600 1.5rem/1 'Cormorant Garamond';
```

## Animations

### Motion Philosophy

Animaciones que evocan la **suavidad orgánica** del estilo bohemio:

| Type   | Duration | Easing      | Uso                 |
| ------ | -------- | ----------- | ------------------- |
| Hover  | 300ms    | ease-out    | Buttons, cards      |
| Expand | 500ms    | ease-out    | Modals, drawers     |
| Fade   | 300ms    | ease        | Transitions         |
| Float  | 6s       | ease-in-out | Decorative elements |
| Pulse  | 2-3s     | ease-in-out | Stars, highlights   |

### Best Practices

- Usar `stone.800` o `stone.900` para texto importante sobre amber
- Nunca usar amber puro como color de texto
- Todos los iconos tienen label accesible
- Focus states visibles con ring amber

## Do's and Don'ts

### Do's ✅

- Usar serif (Cormorant) para títulos grandes
- Combinar amber con purple para contraste
- Mantener espaciado generoso
- Usar sombras suaves con tinte cálido
- Incluir elementos celestiales sutilmente

### Don'ts ❌

- No usar colores neón o saturados extremos
- No mezclar más de 3 fuentes
- No usar esquinas completamente cuadradas
- No abusar de animaciones
- No usar negro puro (#000000)
