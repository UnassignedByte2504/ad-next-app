# Ayla Designs Fonts

This folder contains the brand fonts for Ayla Designs.

## Font Families

### Cormorant Garamond (Headings)
- Use for: H1-H6 headings, product titles, prominent text
- Weights: 400 (Regular), 500 (Medium), 600 (SemiBold), 700 (Bold)
- Includes: Italic for emphasis and quotes

### Nunito Sans (Body)
- Use for: Body text, UI elements, forms, buttons
- Weights: 300 (Light), 400 (Regular), 500 (Medium), 600 (SemiBold)

### JetBrains Mono (Code)
- Use for: Prices, discount codes, technical data
- Weight: 400 (Regular)

## Installation

### Web (CSS)
```css
@font-face {
  font-family: 'Cormorant Garamond';
  src: url('./cormorant-garamond/cormorant-garamond-400.woff2') format('woff2');
  font-weight: 400;
  font-style: normal;
  font-display: swap;
}
```

### Google Fonts (Alternative)
```html
<link href="https://fonts.googleapis.com/css2?family=Cormorant+Garamond:ital,wght@0,400;0,500;0,600;0,700;1,400&family=Nunito+Sans:wght@300;400;500;600&family=JetBrains+Mono:wght@400&display=swap" rel="stylesheet">
```
